# RQY1102_GRUPO_C_ET
# Nombre Proyecto: 
Sistema de gestión de Pago para Edificio Mirador

# Integrantes: 
Brandon Cantero, Amaro Lazo, Sebastián Soto

# Descripción del problema 
El Edificio Mirador presenta problemas administrativos y de gestión de gastos comunes, lo que aumenta el riesgo de impagos, retrasos en mantenciones y afecta la estabilidad financiera y la seguridad de los residentes.

# Solución
Ecosistema digital que automatiza el cálculo de gastos comunes, el registro de pagos, la identificación de morosidad y la comunicación con residentes, aportando transparencia y eficiencia a la administración del edificio.
